This is a stage (extension) for Chub.ai; PARC is a game the runs within a Chub chat: https://chub.ai/characters/JakeH/post-apocalypse-rehabilitation-center-20622f03e7a4 . It is a space station management game with visual novel elements that has the player resurrecting characters from various collapsed universes to give them a second life in a new galaxy. Mechanically, the game pulls character content from the Chub website to give every player a unique experience.
